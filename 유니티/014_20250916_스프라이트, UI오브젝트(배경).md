## 2D

- 스프라이트를 사용한 오브젝트 중 UI를 제외한 오브젝트

## UI

- User Interface의 약자
- 사용자와 프로그램 사이에서 의사소통을 할 수 있도록 만들어진 매개체
  > 캐릭터를 조작하는 조이스틱   
  > 사운드를 조절하는 슬라이더
- Image 컴포넌트를 통해 스프라이트를 그림
