- Dynamic Friction
  - 동적인 상태의 마찰계수
  - 0~1 사이의 값으로 설정
  - 0
    - 마찰이 없음(얼음 위를 미끄러지는 것과 같은 상태)
  - 1
    - 마찰이 강해져서 미끄러지기 힘든 상태
- Static Friction
  - 정적인 상태의 마찰계수
  - Dynamic Fricition과 같은 범위와 상태
- Bounciness
  - 오브젝트가 물리적으로 부딪힐 때 반발하는 계수(반발계수)
  - 0
    - 전혀 튀어 오르지 않음
  - 1
    - 크게 튀어 오름
- Friction Combine
  - 두 오브젝트가 물리적으로 부딪힐 때의 마찰계수 계산 방법 지정
  - Average
    - 두 오브젝트의 계수 평균값 적용
  - Multiply
    - 두 오브젝트의 계수 곱 적용
  - Minimum
    - 두 오브젝트 중 계수가 작은 쪽의 값 적용
  - Maximum
    - 두 오브젝트 중 계수가 큰 쪽의 값 적용
- Bounce Combine
  - 두 오브젝트가 물리적으로 부딪힐 때의 반발계수 계산 방법 지정
  - Friction Conbine과 설정 종류가 같음

- Shader
  - 화면에 표시할 규칙 설정
  - 아래 항목들은 Standard 사용시 설정 가능한 항목
- Rendering Mode
  - 투명도
  - Opaque
    - 불투명
  - Cutout
    - 일부만 투명
    - 이미지(텍스처) 데이터를 사용하여 화면에 표시할 때 이미지의 일정 투명도 이하 부분은 화면에 표시하지 않음
  - Fade
    - 투명 표현을 페이드인/페이드아웃하는 것처럼 보이게 함
  - Transparent
    - 텍스처의 투명도에 따라 투명
- Albedo
  - 물질이 빛을 반사할 때의 색 설정(기본적인 외관)
- Metallic
  - 물질의 표면이 얼마나 금속적인지 설정
  - 금속적일수록 주변의 색을 반사하는 정도가 높아짐
- Normal Map
  - 법선 맵
  - 울퉁불퉁함을 표현하기 위해 특수한 이미지 설정
- Height Map
  - Normal Map보다 더 큰 값의 굴곡을 표현할 수 있는 이미지 설정
- Occlusion
  - 물질의 일부분이 얼마만큼 간접 조명의 영향을 받는지 표현하기 위한 이미지 설정
  - 덮개 등으로 빛을 차단한 물체가 있을 때 덮개로 인해 어두워진 부분 표현
- Emission
  - 자체 발광에 대한 색과 세기 설정 
- Tiling
  - 이미지 반복 횟수
- Offset
  - 이미지를 어긋나게 표시하기 위한 설정정
