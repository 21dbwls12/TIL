# Canvas 컴포넌트

## Render Mode

### Screen Space - Overlay

- 가장 앞쪽에 표시

Pixel Perfect|UI가 가장 확실하게 보이는 상태로 설정
|:---:|:---|
Sort Order|값이 가장 큰 Canvas를 가장 앞쪽에(Canvas가 여려개 있을 때)
Target Display|게임 안에 카메라가 여러 대 있을 때 그리는 카메라 전환(카메라에서 Target Display 설정)
Additional Shader Channels|UI에 셰이더 채널 추가

<br>

### Screen Space - Camera

- 씬을 표시하는 카메라 외에 UI를 표시하는 카메라 추가 가능
  - 카메라의 그리기 순서를 조절하면 오브젝트가 UI보다 앞에 보이게 설정 가능

Pixel Perfect|UI가 확실하게 보이는 상태로 설정
|:---:|:---|
Render Camera|UI용 카메라 설정
Plane Distance|카메라와 Canvas의 거리 지정
Sorting Layer|Canvas가 여러 개 있을 때 그룹으로 분리
Order in Layer|그리기 순서 설정(값이 큰 그룹을 앞쪽에)
Additional Shader Channels|UI에 셰이더 채널 추가

<br>

#### Camera 설정

- Culling Mask
  - UI
    - UI가 설정된 오브젝트만 Layer에 표시
- Clear Flags
  - Depth only
    - Depth 값이 큰 카메라를 앞쪽에 표시


<br>

### World Space

- Canvas의 위치와 크기를 자유롭게 설정 가능
  - Rect Transform
    - 위치와 크기

Event Camera|UI에 어떤 처리를 하기 위한 설정 항목(ex.버튼을 마우스로 클릭)
|:---:|:---|
Sorting Layer|Canvas가 여러 개 있을 때 그룹으로 분리
Order in Layer|그리기 순서 설정(값이 큰 그룹을 앞쪽에)
Additional Shader Channels|UI에 셰이더 채널 추가
