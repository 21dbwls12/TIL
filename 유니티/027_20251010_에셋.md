# 에셋 정리

## 에셋 내 파일 종류

- 씬(Scene)
  - 에셋의 사용법을 담은 미리보기용 씬
- 모델(Model, fbx)
  - 3D 모델링 파일
- 프리팹(Prefab)
  - 바로 사용이 가능하도록 만들어둔 프리팹
- 머터리얼(Material)
  - 오브젝트에 사용된 머터리얼
- 텍스처(Texture)
  - 머터리얼에 사용된 이미지
- 문서(ReadMe)
  - 에셋의 버전이나 설명이 담긴 문서

<br>

## 에셋 정리

### 삭제하면 안되는 파일

- 모델
- 머터리얼
- 텍스처

<br>

### 삭제해도 되는 파일

- 연결을 끊은 프리팹
- 씬 파일(미리보기용)
- 설명 문서
