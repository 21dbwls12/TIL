()는 앞에와 붙여쓰고 숫자 int도 사용해보라는 피드백을 받았다.

```java
class HelloWorld {
    public static void main(String[] args) {
     String name = "Choi Yoojin";
     String birth = "1998.11.17";
     int checkin = 7;
     String experient = "none";
     System.out.print(name);
     System.out.print(", ");
     System.out.println(birth);
     System.out.println(checkin);
     System.out.print(experient);
    }
}
```
