### 문제 설명

<p>문자를 입력받으면 아스키코드 10진수 int로 전환하고 0~127 사이의 int 값을 입력받으면 아스키코드에 해당하는 문자로 전환하는 함수를 작성하세요.</p>
<p>'A' -> 65</p>
<p>97 -> 'a'</p>
<p>'4' -> 52</p>
<p>48 -> '0'</p>

### 입력

 <p>0 <= N <= 127인 Int 또는 Char</p>
