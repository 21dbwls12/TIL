### 문제 설명

<p>정렬되지 않은 주어진 정수 배열에서 2번째로 큰 값을 찾는 함수를 작성하세요.</p>

### 입력

 <p>첫째 줄에 여러개의 정수가 쉼표(,)로 구분된 배열이 주어진다.</p>

### 출력 

 <p>첫째 줄에 입력으로 주어진 배열의 정수 중에 2번째로 큰 값을 출력한다.</p>

### 예제

[5,1,52,57,13,53] >> 53
