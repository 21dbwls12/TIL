## n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.


      첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

      1부터 n까지 합을 출력한다.

      - 3   -> 6

---------------------------
성공  
![image](https://github.com/21dbwls12/TIL/assets/139525941/e0c04fdb-055d-41ae-9cdb-2158cead06c5)

```kotlin
fun main() {
    val n = readln().toInt()
    var sum = 0

    for (i in 1..n) {
        sum += i
    }
    print(sum)
}
```
