## 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

    입력은 여러 개의 테스트 케이스로 이루어져 있다.  
    각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

    각 테스트 케이스마다 A+B를 출력한다.

    - 1 1 -> 2
      2 3    5
      3 4    7
      9 8    17
      5 2    7

------------------
실패  
어떻게 결과값이 나오는 지 확인할 수 없음...(메인보드 이슈)  
일단 미루기..

```kolin
fun main() {
    while (true) {
        val test = readLine()
        
        if (test.isNullOrBlank()) {
            val list = test!!.split(" ")
            val a = list[0].toInt()
            val b = list[1].toInt()
            
            println(a + b)
        } else {
            break
        }
    }
}
```
