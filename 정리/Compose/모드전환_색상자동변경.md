values폴더 안 colors폴더에 colors.xml(night)파일 생성

colors.xml안 컬러값과 같은 이름으로 colors.xml(night)에 컬러값 생성

---------------
### colors.xml
```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
  <color name = "background">@color/white</color>
```

### colors.xml(night)
```xml

<?xml version="1.0" encoding="utf-8"?>
<resources>
  <color name = "background">#FF121212</color>
```
